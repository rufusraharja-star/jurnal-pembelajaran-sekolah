# 📓 Jurnal Pembelajaran Digital SMA
Sistem pencatatan jurnal mengajar harian berbasis web untuk **Sekolah SMA**. Aplikasi ini dirancang untuk menggantikan jurnal fisik menjadi digital agar lebih efisien, rapi, dan mudah direkap.

## ✨ Fitur Utama
- **Pemilihan Kelas Cepat:** Tersedia tombol shortcut untuk Kelas Reguler (X, XI, XII) dan Kelas MPP (A/B/C).
- **Integrasi Nama Guru & Mapel:** Database nama guru dan mata pelajaran sudah tersemat untuk meminimalisir kesalahan input.
- **Preview Kartu Jurnal:** Menampilkan format kartu jurnal resmi yang siap cetak (Print-Ready).
- **Export to Excel (CSV):** Fitur untuk mengunduh seluruh riwayat jurnal ke format spreadsheet untuk keperluan laporan bulanan.
- **Responsive Design:** Dapat diakses dengan nyaman melalui Laptop, Tablet, maupun Smartphone.

## 🚀 Teknologi yang Digunakan
- **HTML5 & CSS3:** Struktur dan styling kustom.
- **Bootstrap 5:** Framework CSS untuk tampilan yang modern dan responsif.
- **Vanilla JavaScript:** Logika aplikasi, pengolahan data, dan sistem ekspor file.
- **LocalStorage:** Penyimpanan data lokal di browser pengguna (tanpa perlu database server eksternal).

## 📂 Struktur File
- `index.html` - Halaman utama / Dashboard rekap data.
- `form.html` - Halaman pengisian jurnal harian.
- `script.js` - Logika sistem dan manajemen data.
- `style.css` - Custom styling dan pengaturan tampilan cetak.

## 🛠️ Cara Penggunaan Lokal
1. Download atau `git clone` repository ini.
2. Buka file `index.html` di browser pilihan Anda (Chrome/Edge/Safari).
3. Pilih kelas, isi data, dan simpan.
4. Gunakan tombol **Export Excel** untuk mengunduh rekapitulasi.
